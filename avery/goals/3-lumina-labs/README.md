# Lumina Labs - Agent 开发框架

## 总体愿景

通过构建三个不同自主程度的 Agent，探索 AI 自主能力边界：
1. **Nova** - 全自主 Agent
2. **Avery** - 半自主助理（本仓库）
3. **Dexter** - 研发任务管理专家

## 目标

- 探索 AI Agent 的自主能力边界
- 建立可复用的 Agent 开发框架
- 积累 Agent 设计和开发经验
- 为求职提供技术亮点
