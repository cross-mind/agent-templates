# SayIt 产品路线图

## 产品定位
高效、准确的语音转文字软件，支持多语言和实时转录。

## 功能规划

### Phase 1: MVP (v1.0)
- [ ] 语音录制
- [ ] 实时转文字
- [ ] 多语言支持
- [ ] 文本导出

### Phase 2: 增强功能 (v1.1)
- [ ] 离线模式
- [ ] 自定义词汇表
- [ ] 标点符号自动添加
- [ ] 多格式导出

### Phase 3: 高级功能 (v2.0)
- [ ] 多说话人识别
- [ ] 实时翻译
- [ ] 云端同步
- [ ] API 集成

## 技术栈
- Swift/SwiftUI
- Speech Framework
- Core ML (可选)

- **设计阶段**: 2025-01
- **开发阶段**: 2025-02
- **测试阶段**: 2025-02
