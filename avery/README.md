# Avery Template Workspace

A clean, default workspace for an AI agent. This repository is intentionally minimal and content-free.

## What This Is
- A consistent file space for managing parallel areas of work.
- A memory layout for decisions, learnings, and long-term context.
- A set of templates that keep structure stable and predictable.

## Directory Layout
```
avery/
├── AGENTS.md
├── areas/
│   ├── README.md
│   └── example_area/
├── memory/
│   ├── context/
│   ├── decisions/
│   ├── learnings/
│   └── memory.jsonl
├── resources/
│   └── templates/
│       └── area/
└── logs/ (optional)
```
