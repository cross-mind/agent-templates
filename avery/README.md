## 📚 使用指南

### 日常工作流

1. **晨间启动** - 检查 `inbox/today.md`，规划今日任务
2. **任务执行** - 协助完成各项任务，实时记录
3. **傍晚总结** - 生成当日日志，更新进度
4. **周五复盘** - 生成周总结，调整下周计划

### 记忆管理

- **重要决策** → `memory/decisions/`
- **学到知识** → `memory/learnings/`
- **长期上下文** → `memory/context/`
- **重要对话** → `memory/conversations/`

### 目标追踪

每周检查三个目标的进度，更新相应的 `progress.md` 文件。
